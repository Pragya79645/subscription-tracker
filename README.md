## Project Overview:
This is a RESTful API backend application that helps users track and manage their subscription services. The system automatically sends email reminders for upcoming subscription renewals using scheduled workflows, ensuring users never miss important payment dates.

## Key Features:

User authentication & authorization (JWT-based)
CRUD operations for subscription management
Automated email notifications for renewal reminders
Security protection with rate limiting
Workflow automation for scheduled tasks
Multi-currency support (USD, EUR, GBP, INR, JPY)

## Tech Stack for Resume:
Backend: Node.js, Express.js, MongoDB, Mongoose ODM
Authentication: JWT (JSON Web Tokens), bcryptjs
Email Services: Nodemailer
Workflow Automation: Upstash Workflow
Security: Arcjet (rate limiting & protection)
Development Tools: ESLint, Nodemon
Additional: RESTful API design, Cookie-based authentication, Error handling middleware
